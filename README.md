# Nexus Landing Page




Projeto de estudos desenvolvido com HTML, CSS e Bootstrap, focado na criação de uma landing page moderna, responsiva e com identidade visual corporativa.

📖 Sobre o projeto

A Nexus Landing Page foi criada com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, explorando estruturação semântica, estilização moderna e utilização de componentes responsivos do Bootstrap.

O projeto apresenta uma interface limpa e profissional para uma empresa fictícia de tecnologia, contendo:

Navbar responsiva
Sessão principal (Hero Section)
Cards de funcionalidades
Área institucional “Quem Somos”
Rodapé completo com informações e contatos
Ícones utilizando Bootstrap Icons

🚀 Tecnologias utilizadas

HTML5
CSS3
Bootstrap 5
Bootstrap Icons

🎯 Objetivo

Este projeto foi desenvolvido exclusivamente para fins de estudo e aprimoramento de habilidades em desenvolvimento web front-end.

Entre os principais conceitos praticados estão:

Estruturação de layouts
Responsividade
Utilização de componentes Bootstrap
Organização visual de landing pages
Boas práticas de HTML e CSS
